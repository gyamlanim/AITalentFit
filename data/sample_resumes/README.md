# Sample Resumes (Synthetic Data)

All resumes in this folder are **AI-generated / synthetic**. The names, contact
details (`(555)` numbers, `@example.com` / demo emails), employers, and work
histories are fictional and do not correspond to real people. They exist to
let you try the app end-to-end without needing real resumes.

Files:
- `ArjunResume-AIGEN.pdf` — synthetic Software Engineer profile
- `DanielRESUMEAIgen.pdf` — synthetic Marketing Specialist profile
- `PriyaSharma-AIGEN.pdf` — synthetic Data Analyst profile
- `MarcusBell-AIGEN.pdf` — synthetic Product Manager profile
- `SofiaAlvarez-AIGEN.pdf` — synthetic Software Engineer profile

Upload any of these through the app's file uploader to see resumes get
parsed, embedded, and ranked against a job description.
